# Contributing to SpeedMate

Thanks for contributing!

## Ground rules
- Keep changes focused and minimal.
- Follow PSRâ€‘12 and WordPress coding standards where applicable.
- Ensure all inputs are sanitized and outputs escaped.
- Avoid loading assets outside the SpeedMate admin page.

## How to contribute
1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Add/adjust tests or notes if needed
5. Open a PR with a clear description

## Reporting issues
Use GitHub Issues with:
- WordPress + PHP versions
- Steps to reproduce
- Expected vs actual behavior

## Security
Please do not open public issues for security vulnerabilities. Use private disclosure (email to be added).
